## echo 与特殊符号
echo命令是我们学习Shell编程一个很常用的命令，他用于打印信息到标准输出，譬如。
```
echo "hello world"    # 打印hello world到标准输出
```
目前echo有两个参数：
* -n 不解析参数内的特殊符号
* -e 默认值，解析参数内的特殊符号

```
echo -n "hello\tworld"    # 不解析参数内制表符，同时不执行echo后的\n,特殊符号见表1
echo -e "hello\tworld"    # 解析参数中的制表符。
```

特殊符号对照表，表1。

| 符号 | 含义 |
| :--- | :--- |
| \n | 新行 |
| \r | 回车 |
| \t | 制表符 |
| \v | 垂直的制表符 |
| \b | 后退符 |
| \a | 警告（蜂鸣或是闪动） |
